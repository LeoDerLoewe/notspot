2. Welche Daten wir erheben

Technisch notwendig (ohne Consent):

  • Dein Standort (nur wenn du ihn freiwillig angibst) – wird nur lokal im Browser gespeichert, nie an uns übertragen
  • Spin-Historie, Filter-Einstellungen, Bewertungen – nur lokal im Browser (localStorage)

Nur mit deiner Zustimmung (Analytics):

  • Keine Weitergabe an Dritte, keine Werbung

3. OpenStreetMap

Für die Karte und Live-POI-Daten werden Anfragen an die Overpass API (openstreetmap.org) gesendet. Dabei werden deine Koordinaten des angezeigten Ortes übermittelt. Es gelten die Datenschutzbestimmungen der OSMF.

4. Netlify (Hosting)

Die Website wird über Netlify (USA) gehostet. Netlify verarbeitet Server-Logs nach eigenen Datenschutzrichtlinien. Für Wartelisten-Formulare gelten Netlify's Datenschutzbestimmungen.

5. Deine Rechte

  • Auskunft, Berichtigung, Löschung deiner Daten
  • Widerruf der Analytics-Einwilligung jederzeit (Cookie-Banner erneut anzeigen: Seite neu laden, Banner erscheint nach 30 Tagen wieder)
  • Kontakt: info@notspot.ch

6. Datenspeicherung

Deine lokalen Daten (Spin-Historie, Standort, Einstellungen) werden ausschliesslich in deinem Browser gespeichert und nie an unsere Server übertragen. Du kannst sie jederzeit löschen: Browser → Einstellungen → Browserdaten löschen.
